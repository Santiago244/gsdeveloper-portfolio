The first projects do not need to be huge. They need to be clear, finished, and easy to explain.

## Project One: Portfolio Site

This site can be the first case study. It shows Astro, design customization, content structure, and Cloudflare deployment.

## Project Two: API Dashboard

A focused dashboard can show async JavaScript, API consumption, loading states, error handling, and thoughtful UI states.

## Project Three: Interactive Visual

A small Three.js experiment can show motion and creative frontend work. It should stay focused: one polished interaction is stronger than a large unfinished scene.
